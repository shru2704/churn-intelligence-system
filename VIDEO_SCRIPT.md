## Screen Recording Instructions

### Option 1: Loom (Recommended)

1. Go to https://loom.com
2. Download Loom desktop app
3. Click "New Recording"
4. Select "Screen + Camera"
5. Open your dashboard: `http://localhost:8501` (or live URL)
6. Click record, follow script
7. Share link

### Option 2: Zoom

1. Start Zoom meeting
2. Click "Share Screen"
3. Check "Share computer sound"
4. Record to cloud
5. Download MP4

### Option 3: Microsoft Teams

1. Start meeting with yourself
2. Click "Share" â†’ "Screen"
3. Start recording
4. Download when done

### Option 4: OBS Studio (Free)

1. Download OBS Studio
2. Add Display Capture
3. Add Microphone
4. Start Recording
5. Edit in CapCut (free) if needed

## Practice Tips

| Tip | Why |
|-----|------|
| **Record 2-3 takes** | Pick the best one |
| **Speak slowly** | Nervousness makes you rush |
| **Smile at start/end** | Makes you more approachable |
| **Gesture at screen** | Directs viewer attention |
| **Pause between sections** | Easier to edit later |

---

## Editing Quick Wins (Optional)

Use CapCut (free) or iMovie:

| Edit | Time Saved |
|------|------------|
| Trim dead air at start/end | 5 seconds |
| Cut "umms" and pauses | 10-20 seconds |
| Add text overlays (key stats) | Professional polish |
| Add background music (low volume) | Engagement boost |

## Common Mistakes to Avoid

| Mistake | Fix |
|---------|-----|
| Talking too fast | Practice with a timer |
| Reading from script verbatim | Use bullet points, sound natural |
| Screen too small | Zoom in on relevant sections |
| Audio echo | Use headphones |
| Forgetting to show live URL | Add URL in video description |
